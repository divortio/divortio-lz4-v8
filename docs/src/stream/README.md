The **Stream** module implements the [Web Streams API](https://developer.mozilla.org/en-US/docs/Web/API/Streams_API) for LZ4. It provides `TransformStream` factories for streaming usage.

## Files
-   [**streamCompress**](streamCompress.md): Standard compression stream.
-   [**streamDecompress**](streamDecompress.md): Standard decompression stream.
-   [**streamAsyncCompress**](streamAsyncCompress.md): Time-sliced compression stream.
-   [**streamAsyncDecompress**](streamAsyncDecompress.md): Time-sliced decompression stream.
-   [**scheduler**](scheduler.md): Async task scheduler.
