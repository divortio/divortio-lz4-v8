The **CLI** module provides the logic for the Command Line Interface. It handles argument parsing, configuration, file I/O coordination, and logging.

## Files
-   [**cliArgs**](cliArgs.md): Argument parser.
-   [**cliConfig**](cliConfig.md): Configuration state / validator.
-   [**cliCompress**](cliCompress.md): Compression controller.
-   [**cliDecompress**](cliDecompress.md): Decompression controller.
-   [**cliHelp**](cliHelp.md): Help text generator.
-   [**cliLog**](cliLog.md): Structured logging (JSON/CSV).
-   [**cliUtils**](cliUtils.md): Formatters.
