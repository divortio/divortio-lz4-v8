## Description
Pure utility functions for string formatting and math within the CLI.

## Functions
-   `formatBytes`: Human readable sizes (MB, GB).
-   `formatDuration`: ms to string.
-   `formatThroughput`: Bytes/Time -> MB/s.
-   `formatRatio`: Size diff -> Percentage.
