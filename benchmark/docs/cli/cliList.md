# CLI Command: List

General listCorpora command. Dispatches to specific listCorpora types.

## Usage

```bash
node benchmark/bench.js listCorpora <type> [options]
```

## Types

-   `libs`, `libraries`: List libraries.
-   `corpus`, `corpora`: List corpus files.

See [cliListLibs.md](./cliListLibs.md) and [cliListCorpus.md](./cliListCorpus.md) for details.
