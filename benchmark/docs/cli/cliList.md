# CLI Command: List

General list command. Dispatches to specific list types.

## Usage

```bash
node benchmark/bench.js list <type> [options]
```

## Types

-   `libs`, `libraries`: List libraries.
-   `corpus`, `corpora`: List corpus files.

See [cliListLibs.md](./cliListLibs.md) and [cliListCorpus.md](./cliListCorpus.md) for details.
